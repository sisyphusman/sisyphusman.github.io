- Visual Studio + vcpkg + CMake + SFML을 사용해서 2D 로그라이크 만들기

### vcpkg

- vcpkg에서 패키지 설치를 진행할때 CMake 코드에서 인코딩 버그가 있음 (libflac 패키지 등)  
윈도우의 시스템 언어 설정을 영어로 임시 변경 후 설치 진행할 것

1. vcpkg 설치
   1. git clone https://github.com/microsoft/vcpkg.git
   2. cd vcpkg
   3. .\bootstrap-vcpkg.bat
   4. .\vcpkg integrate install
2. vcpkg 환경 변수 등록
3. SFML을 vcpkg로 설치
   1. vcpkg install sfml:x64-windows
   
### CMake

1. 프로젝트 폴더 + src/main.cpp + CMakeLists.txt 생성
2. CMake GUI에서 source/build 폴더 지정
3. CMAKE_TOOLCHAIN_FILE에 vcpkg.cmake 설정
4. Configure → Generate → VS로 열기
5. VS에서 빌드 & 실행

### CMakeLists.txt

```
cmake_minimum_required(VERSION 3.20)

project(project-1 LANGUAGES CXX)

# C++ 버전 설정
set(CMAKE_CXX_STANDARD 20)
set(CMAKE_CXX_STANDARD_REQUIRED ON)

# 실행 파일 추가
add_executable(project-1
	src/main.cpp
)

# vcpkg
find_package(SFML 3 CONFIG REQUIRED COMPONENTS
    System
    Window
    Graphics
    Audio
    Network
)

# SFML3
target_link_libraries(project-1 PRIVATE
    SFML::System
    SFML::Window
    SFML::Graphics
    SFML::Audio
    SFML::Network
)
```
